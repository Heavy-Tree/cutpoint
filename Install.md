# INSTALL.md

## Установка и запуск проекта CutPoint

### Требования

Требования по софту перед запуском:

* Python 3.11 или новее
* Node.js 18.x или новее
* MySQL Server 8.0 или новее
* Git (для клонирования)

---

## 1. Клонирование проекта

git clone https://github.com/Heavy-Tree/cutpoint.git
cd cutpoint

Либо распакуйте архив проекта в любую папку.

---

## 2. Настройка базы данных

При первом запуске автоматически создаются необходимые таблицы.
При необходимости можно удалить старую базу и создать её заново.

---

## 3. Настройка бэкенда

Перейдите в папку backend:

cd backend

Создайте виртуальное окружение:

Windows:

python -m venv venv
venv\Scripts\activate

Mac/Linux:

python3 -m venv venv
source venv/bin/activate

Установите зависимости:

pip install -r requirements.txt

Создайте файл .env и укажите подключение к MySQL:

DATABASE_URL=mysql+pymysql://ЛОГИН:ПАРОЛЬ@localhost:3306/cutpoint
JWT_SECRET=супер-секретный-ключ

где:

* ЛОГИН — имя пользователя MySQL
* ПАРОЛЬ — пароль пользователя MySQL

Запустите бэкенд:

uvicorn app.main:app --reload --port 8000

Бэкенд будет доступен по адресу:

http://127.0.0.1:8000

Документация API:

http://127.0.0.1:8000/docs

---

## 4. Настройка фронтенда

Откройте новый терминал и перейдите в папку frontend:

cd frontend

Установите зависимости:

npm install

Запустите фронтенд:

npm run dev

Фронтенд будет доступен по адресу:

http://127.0.0.1:5173

---

## 5. Использование

На сайте доступны следующие возможности:

* регистрация пользователей;
* авторизация;
* просмотр каталога ножей;
* фильтрация и сортировка товаров;
* пагинация;
* просмотр детальной карточки ножа;
* добавление ножей в избранное;
* страница избранного;
* сравнение ножей;
* админ-панель (добавление/удаление ножей);
* личный кабинет;
* поиск по каталогу.

---

## 6. Возможные проблемы

Ошибка подключения к базе данных:

* Убедитесь, что MySQL сервер запущен
* Проверьте логин и пароль в файле .env

Ошибка при запуске бэкенда:

* Проверьте версию Python (должна быть 3.11+)
* Переустановите зависимости: pip install -r requirements.txt

Ошибка при запуске фронтенда:

* Удалите папку node_modules и файл package-lock.json, затем выполните npm install заново

Порт уже используется:

* Измените порт в команде запуска (--port 8001) либо завершите процесс, использующий данный порт.
